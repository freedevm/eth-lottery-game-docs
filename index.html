Ethereum Lottery Game Documentation

Your guide to understanding and playing the decentralized lottery

Overview

The Ethereum Lottery Game is a decentralized application (dApp) built on the Ethereum blockchain. Players can participate by purchasing tickets using Ether (ETH), and winners are selected randomly via a secure smart contract. The game ensures transparency, fairness, and immutability through blockchain technology.

Smart Contract Details

The lottery is powered by a Solidity smart contract deployed on the Ethereum blockchain. Below are key details:

  • Prize Distribution: 70% to winner, 20% to next round, 10% to admin

Smart Contract Code

pragma solidity ^0.8.0;
import "@chainlink/contracts/src/v0.8/VRFConsumerBase.sol";

contract Lottery is VRFConsumerBase {
    address public admin;
    uint256 public ticketPrice = 0.01 ether;
    address[] public players;
    
    constructor(address _vrfCoordinator, address _link) 
        VRFConsumerBase(_vrfCoordinator, _link) {
        admin = msg.sender;
    }
    
    function buyTicket() public payable {
        require(msg.value == ticketPrice, "Incorrect ticket price");
        players.push(msg.sender);
    }
    
    // Additional functions for randomness and prize distribution
}

How to Play

  1. Connect your Ethereum wallet (e.g., MetaMask) to the dApp.
  2. Ensure you have sufficient ETH to cover ticket price and gas fees.
  3. Purchase a ticket by sending 0.01 ETH to the contract.
  4. Wait for the lottery round to end and the winner to be selected.
  5. Check the results on the dApp or Etherscan.

FAQ - Fairness

Is the lottery fair?

Yes, the lottery uses Chainlink VRF to ensure provably fair randomness, and all transactions are transparent on the blockchain.

FAQ - Rollover

What happens if no one wins?

If no winner is selected (e.g., not enough players), the prize pool rolls over to the next round.

FAQ - Gas Fees

How are gas fees handled?

Players are responsible for gas fees when purchasing tickets. Ensure your wallet has enough ETH.
